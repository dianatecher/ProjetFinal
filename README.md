# Projet final de Simplon Promo #4
## Choix: blog

### Let's get started
```shell
$ npm install
```

### Commands
Development:
Webpack-server-dev with watching functionality:
  ```shell
  $ npm run webpack-server
  ```
  Start nodemon:
  ```shell
  $ npm run start
  ```

Production:
  ```shell
  $ npm run build
  ```
