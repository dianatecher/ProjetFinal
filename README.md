# Projet final de Simplon Promo #4
## Choix: blog

### Let's get started
```shell
$ npm install
```

### Commands
Development:
Webpack-server-dev with watching functionality (on localhost:8080):
	```shell
	$ npm run webpack-server
	```

Start nodemon (on localhost:3000):
	```shell
	$ npm run start
	```

Production:
	```shell
	$ npm run build
	```
